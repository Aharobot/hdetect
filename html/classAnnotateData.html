AnnotateData::AnnotateData ( std::string  bagFile  ) 

Sets up everything to read and annotate the given bag. First it creates some fake subscribers for each one of the camera, camera info and laser topics. The it sets up the approximate time policy synchronizer. Then it uses bagger, a bagReader class object, to open and feed the subscribers with messages from the topics. When all three messages arrive to the corresponding subscribers then the synchronizer calls the annotateData() function of the annotator object. It goes on with doing the annotation

Parameters:
[in] bagFile The name of the bag file to be annotated
